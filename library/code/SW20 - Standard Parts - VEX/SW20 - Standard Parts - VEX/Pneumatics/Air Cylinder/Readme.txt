When using air cyclinder in standard part lib, please copy it to your own folder instead of using it directly from lib.

Three things to do before using:
a) Change configuration of Cylinder Rod, select "All configuration";
b) Change configuration of Cylinder Body, select "All configuration";
c) Change "tool"-"function"-"global variable"-"Cyclinder Stroke";

if you want the rod to move in a limit distance, do as follow:
a) Unsuppress "Rod Distance Limit";
b) Clarify the Max Dist;
